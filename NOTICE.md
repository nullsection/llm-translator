# Credits & licenses

This tool bundles several open-source components. All run fully offline and are
free to use. One component (VOICEVOX, for Japanese speech) asks for a credit line
when you **publicly share** its audio.

## VOICEVOX (Japanese text-to-speech)

Japanese speech is produced by **VOICEVOX**. It is free for personal **and**
commercial use, with **no fee and no account**. The voice used by default is
**四国めたん (VOICEVOX)**.

> When you publicly share audio generated by this tool's Japanese voice
> (videos, posts, products, etc.), display a credit such as:
>
> **VOICEVOX:四国めたん**

For private/personal use (listening yourself), no action is required. Full terms
are included with the downloaded assets under `models/voicevox`.

## NLLB-200 (high-quality translation model)

Translation uses Meta's **NLLB-200-distilled-1.3B** (via CTranslate2). The model
weights are licensed **CC-BY-NC 4.0 — non-commercial**. Free for personal use;
**not** permitted in a commercial product. For commercial use, switch to the
unrestricted Argos backend: set `TRANSLATOR_MT=argos` and run
`translator install en` / `ja` / `zh` once (online) to fetch its packages.
